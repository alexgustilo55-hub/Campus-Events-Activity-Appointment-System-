# Project Description
The Campus Events & Activity Appointment System is a web-based platform designed to streamline the management of campus events and student appointments. The system replaces manual processes, which often rely on paper forms and spreadsheets, with an efficient digital solution that ensures accuracy, transparency, and real-time updates.


# Problem Statement
Managing campus events and student appointments is often done manually using paper forms and spreadsheets. This approach is:
- Time-consuming
- Prone to errors
- Lacks transparency between students, organizers, and administrators


# Solution
This system provides:
- A frontend and backend built with Flask
- A MySQL database to store all event and appointment data  
- Organizer functionalities: create and manage events, view student registrants  
- Student functionalities: view available events, register for appointments  
- Admin functionalities: approve or reject student appointments  
- Automatic updating of appointment status in real time  


## Features
- Event creation and management by organizers
- Student registration for events
- Appointment approval/rejection by administrators
- Real-time status updates of appointments
- Secure data storage using MySQL
